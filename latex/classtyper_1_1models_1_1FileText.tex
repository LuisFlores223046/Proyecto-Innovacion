\doxysubsection{Detailed Description}
\begin{DoxyVerb}Gives you a file-like object for reading text, and you will get a `str` data from it.
The default mode of this class is `mode="r"`.

**Example**

```python
from typing import Annotated

import typer

app = typer.Typer()

@app.command()
def main(config: Annotated[typer.FileText, typer.Option()]):
    for line in config:
        print(f"Config line: {line}")

if __name__ == "__main__":
    app()
```
