{\footnotesize\ttfamily \label{classpydantic_1_1v1_1_1utils_1_1ValueItems_a300dec0fd23f357e887b6ff535c1bf22} 
 Any pydantic.\+v1.\+utils.\+Value\+Items.\+merge (\begin{DoxyParamCaption}\item[{}]{cls}{, }\item[{Any}]{base}{, }\item[{Any}]{override}{, }\item[{bool }]{intersect}{ = {\ttfamily False}}\end{DoxyParamCaption})}

\begin{DoxyVerb}Merge a ``base`` item with an ``override`` item.

Both ``base`` and ``override`` are converted to dictionaries if possible.
Sets are converted to dictionaries with the sets entries as keys and
Ellipsis as values.

Each key-value pair existing in ``base`` is merged with ``override``,
while the rest of the key-value pairs are updated recursively with this function.

Merging takes place based on the "union" of keys if ``intersect`` is
set to ``False`` (default) and on the intersection of keys if
``intersect`` is set to ``True``.
\end{DoxyVerb}
