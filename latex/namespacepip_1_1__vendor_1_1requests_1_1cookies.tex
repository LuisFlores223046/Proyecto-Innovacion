\doxysubsection{Detailed Description}
\begin{DoxyVerb}requests.cookies
~~~~~~~~~~~~~~~~

Compatibility code to be able to use `cookielib.CookieJar` with requests.

requests.utils imports from here, so be careful with imports.
\end{DoxyVerb}
