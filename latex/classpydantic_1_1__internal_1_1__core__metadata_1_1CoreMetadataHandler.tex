\doxysubsection{Detailed Description}
\begin{DoxyVerb}Because the metadata field in pydantic_core is of type `Any`, we can't assume much about its contents.

This class is used to interact with the metadata field on a CoreSchema object in a consistent
way throughout pydantic.
\end{DoxyVerb}
