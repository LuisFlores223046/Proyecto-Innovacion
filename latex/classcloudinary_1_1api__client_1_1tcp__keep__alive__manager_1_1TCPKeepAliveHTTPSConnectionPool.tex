\doxysubsection{Detailed Description}
\begin{DoxyVerb}This class overrides the _validate_conn method in the HTTPSConnectionPool class. This is the entry point to use
for modifying the socket as it is called after the socket is created and before the request is made.
\end{DoxyVerb}
