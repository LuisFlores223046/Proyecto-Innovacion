Cookie accepts {\ttfamily encode} or {\ttfamily decode} options for processing a \href{https://datatracker.ietf.org/doc/html/rfc6265\#section-4.1.1}{\texttt{cookie-\/value}}. Since the value of a cookie has a limited character set (and must be a simple string), these functions are used to transform values into strings suitable for a cookies value.

The default {\ttfamily encode} function is the global {\ttfamily encode\+URIComponent}.

The default {\ttfamily decode} function is the global {\ttfamily decode\+URIComponent}, wrapped in a {\ttfamily try..catch}. If an error is thrown it will return the cookie\textquotesingle{}s original value.
