Return an array of all possible and valid expansions of {\ttfamily str}. If none are found, {\ttfamily \+[str]\+} is returned.

The {\ttfamily options} object can provide a {\ttfamily max} value to cap the number of expansions allowed. This is limited to {\ttfamily 100\+\_\+000} by default, to prevent DoS attacks.

A numeric sequence from {\ttfamily x} to {\ttfamily y} inclusive, with optional increment. If {\ttfamily x} or {\ttfamily y} start with a leading {\ttfamily 0}, all the numbers will be padded to have equal length. Negative numbers and backwards iteration work too.


\begin{DoxyCode}{0}
\DoxyCodeLine{;/\string^-\/?\(\backslash\)d+\(\backslash\).\(\backslash\).-\/?\(\backslash\)d+(\(\backslash\).\(\backslash\).-\/?\(\backslash\)d+)?\$/}
\DoxyCodeLine{//\ \{x..y[..incr]\}}

\end{DoxyCode}


An alphabetic sequence from {\ttfamily x} to {\ttfamily y} inclusive, with optional increment. {\ttfamily x} and {\ttfamily y} must be exactly one character, and if given, {\ttfamily incr} must be a number.

For compatibility reasons, the string {\ttfamily \$\{} is not eligible for brace expansion.
