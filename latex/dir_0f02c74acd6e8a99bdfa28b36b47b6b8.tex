\hypertarget{dir_0f02c74acd6e8a99bdfa28b36b47b6b8_autotoc_md821}{}\doxysubsection{\texorpdfstring{Implementation}{Implementation}}\label{dir_0f02c74acd6e8a99bdfa28b36b47b6b8_autotoc_md821}
In order to use this meta stream properly, here are a few things you should know about the implementation.\hypertarget{dir_0f02c74acd6e8a99bdfa28b36b47b6b8_autotoc_md822}{}\doxysubsubsection{\texorpdfstring{Event Buffering /\+ Proxying}{Event Buffering /\+ Proxying}}\label{dir_0f02c74acd6e8a99bdfa28b36b47b6b8_autotoc_md822}
All events of the {\ttfamily source} stream are hijacked by overwriting the {\ttfamily source.\+emit} method. Until node implements a catch-\/all event listener, this is the only way.

However, delayed-\/stream still continues to emit all events it captures on the {\ttfamily source}, regardless of whether you have released the delayed stream yet or not.

Upon creation, delayed-\/stream captures all {\ttfamily source} events and stores them in an internal event buffer. Once {\ttfamily delayed\+Stream.\+release()} is called, all buffered events are emitted on the {\ttfamily delayed\+Stream}, and the event buffer is cleared. After that, delayed-\/stream merely acts as a proxy for the underlaying source.\hypertarget{dir_0f02c74acd6e8a99bdfa28b36b47b6b8_autotoc_md823}{}\doxysubsubsection{\texorpdfstring{Error handling}{Error handling}}\label{dir_0f02c74acd6e8a99bdfa28b36b47b6b8_autotoc_md823}
Error events on {\ttfamily source} are buffered /\+ proxied just like any other events. However, {\ttfamily delayed\+Stream.\+create} attaches a no-\/op {\ttfamily \textquotesingle{}error\textquotesingle{}} listener to the {\ttfamily source}. This way you only have to handle errors on the {\ttfamily delayed\+Stream} object, rather than in two places.\hypertarget{dir_0f02c74acd6e8a99bdfa28b36b47b6b8_autotoc_md824}{}\doxysubsubsection{\texorpdfstring{Buffer limits}{Buffer limits}}\label{dir_0f02c74acd6e8a99bdfa28b36b47b6b8_autotoc_md824}
delayed-\/stream provides a {\ttfamily max\+Data\+Size} property that can be used to limit the amount of data being buffered. In order to protect you from bad {\ttfamily source} streams that don\textquotesingle{}t react to {\ttfamily source.\+pause()}, this feature is enabled by default.\hypertarget{dir_0f02c74acd6e8a99bdfa28b36b47b6b8_autotoc_md825}{}\doxysubsection{\texorpdfstring{API}{API}}\label{dir_0f02c74acd6e8a99bdfa28b36b47b6b8_autotoc_md825}
