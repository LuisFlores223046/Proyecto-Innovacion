\doxysubsection{Detailed Description}
\begin{DoxyVerb}LanguageAlpha2 parses languages codes in the [ISO 639-1 alpha-2](https://en.wikipedia.org/wiki/ISO_639-1)
format.

```py
from pydantic import BaseModel

from pydantic_extra_types.language_code import LanguageAlpha2


class Movie(BaseModel):
    audio_lang: LanguageAlpha2
    subtitles_lang: LanguageAlpha2


movie = Movie(audio_lang='de', subtitles_lang='fr')
print(movie)
# > audio_lang='de' subtitles_lang='fr'
```
