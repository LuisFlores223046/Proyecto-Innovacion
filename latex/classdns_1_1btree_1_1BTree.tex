\begin{DoxyVerb}Create a BTree.

If *original* is not ``None``, then the BTree is shallow-cloned from
*original* using copy-on-write.  Otherwise a new BTree with the specified
*t* value is created.

The BTree is not thread-safe.
\end{DoxyVerb}
