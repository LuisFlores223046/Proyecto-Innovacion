\doxysubsection{Detailed Description}
\begin{DoxyVerb}A Template which is constructed given an existing Python module.

e.g.::

     t = Template("this is a template")
     f = file("mymodule.py", "w")
     f.write(t.code)
     f.close()

     import mymodule

     t = ModuleTemplate(mymodule)
     print(t.render())
\end{DoxyVerb}
