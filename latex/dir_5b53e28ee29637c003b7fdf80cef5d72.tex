\href{https://sonner.emilkowal.ski/}{\texttt{Sonner}} is an opinionated toast component for React. You can read more about why and how it was built \href{https://emilkowal.ski/ui/building-a-toast-component}{\texttt{here}}.\hypertarget{dir_5b53e28ee29637c003b7fdf80cef5d72_autotoc_md2561}{}\doxysubsubsection{\texorpdfstring{Usage}{Usage}}\label{dir_5b53e28ee29637c003b7fdf80cef5d72_autotoc_md2561}
To start using the library, install it in your project:\+


\begin{DoxyCode}{0}
\DoxyCodeLine{npm\ install\ sonner}

\end{DoxyCode}


Add {\ttfamily \texorpdfstring{$<$}{<}Toaster /\+\texorpdfstring{$>$}{>}} to your app, it will be the place where all your toasts will be rendered. After that you can use {\ttfamily toast()} from anywhere in your app.


\begin{DoxyCode}{0}
\DoxyCodeLine{import\ \{\ Toaster,\ toast\ \}\ from\ 'sonner';}
\DoxyCodeLine{}
\DoxyCodeLine{//\ ...}
\DoxyCodeLine{}
\DoxyCodeLine{function\ App()\ \{}
\DoxyCodeLine{\ \ return\ (}
\DoxyCodeLine{\ \ \ \ <div>}
\DoxyCodeLine{\ \ \ \ \ \ <Toaster\ />}
\DoxyCodeLine{\ \ \ \ \ \ <button\ onClick=\{()\ =>\ toast('My\ first\ toast')\}>Give\ me\ a\ toast</button>}
\DoxyCodeLine{\ \ \ \ </div>}
\DoxyCodeLine{\ \ );}
\DoxyCodeLine{\}}

\end{DoxyCode}
