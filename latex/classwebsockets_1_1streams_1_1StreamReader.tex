\doxysubsection{Detailed Description}
\begin{DoxyVerb}Generator-based stream reader.

This class doesn't support concurrent calls to :meth:`read_line`,
:meth:`read_exact`, or :meth:`read_to_eof`. Make sure calls are
serialized.
\end{DoxyVerb}
