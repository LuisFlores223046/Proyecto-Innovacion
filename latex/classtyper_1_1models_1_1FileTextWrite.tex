\doxysubsection{Detailed Description}
\begin{DoxyVerb}You can use this class for writing text. Alternatively, you can use `FileText` with `mode="w"`.
The default mode of this class is `mode="w"`.

**Example**

```python
from typing import Annotated

import typer

app = typer.Typer()

@app.command()
def main(config: Annotated[typer.FileTextWrite, typer.Option()]):
    config.write("Some config written by the app")
    print("Config written")

if __name__ == "__main__":
    app()
```
