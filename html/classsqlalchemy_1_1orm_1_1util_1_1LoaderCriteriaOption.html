Detailed Description

Add additional WHERE criteria to the load for all occurrences of
a particular entity.

:class:`_orm.LoaderCriteriaOption` is invoked using the
:func:`_orm.with_loader_criteria` function; see that function for
details.

.. versionadded:: 1.4
