◆ total_tokens() [1/2]

float anyio._core._synchronization.CapacityLimiter.total_tokens ( self) 
The total number of tokens available for borrowing.

This is a read-write property. If the total number of tokens is increased, the
proportionate number of tasks waiting on this limiter will be granted their
tokens.

.. versionchanged:: 3.0
    The property is now writable.
.. versionchanged:: 4.12
    The value can now be set to 0.

Reimplemented in anyio._core._synchronization.CapacityLimiterAdapter, and anyio._core._synchronization.CapacityLimiterAdapter.
