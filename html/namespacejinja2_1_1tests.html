◆ test_defined()

bool test_defined ( t.Any value) 
Return true if the variable is defined:

.. sourcecode:: jinja

    {% if variable is defined %}
        value of variable: {{ variable }}
    {% else %}
        variable is not defined
    {% endif %}

See the :func:`default` filter for a simple way to set undefined
variables.
