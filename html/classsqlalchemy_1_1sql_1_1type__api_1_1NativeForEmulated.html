Detailed Description

Indicates DB-native types supported by an :class:`.Emulated` type.

.. versionadded:: 1.2.0b3

TypeEngine[Any] sqlalchemy.sql.type_api.NativeForEmulated.adapt_emulated_to_native ( cls,
Union[TypeEngine[Any], TypeEngineMixin] impl,
**Any kw ) 
Given an impl, adapt this type's class to the impl assuming
"native".

The impl will be an :class:`.Emulated` class but not a
:class:`.NativeForEmulated`.

e.g.: postgresql.ENUM produces a type given an Enum instance.

Reimplemented in sqlalchemy.dialects.mysql.enumerated.ENUM, sqlalchemy.dialects.oracle.types.INTERVAL, sqlalchemy.dialects.postgresql.asyncpg.AsyncPgInterval, sqlalchemy.dialects.postgresql.named_types.ENUM, sqlalchemy.dialects.postgresql.pg8000._PGInterval, sqlalchemy.dialects.postgresql.types.INTERVAL, and sqlalchemy.sql.sqltypes.UUID.
